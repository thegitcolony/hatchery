# Contributing

Hatchery is open source. Contributions welcome.

## How to contribute

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Open a pull request

## Ideas we'd love help with

- New creature species with unique shapes
- New biome types
- Sound/music integration
- Mobile touch controls
- Creature breeding algorithms
- Persistent state (localStorage or backend)

## Code style

- Vanilla JS, no frameworks
- Keep it in a single HTML file for now
- Canvas rendering only (no DOM game elements)
- Comment complex logic
