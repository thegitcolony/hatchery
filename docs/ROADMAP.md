# Roadmap

## v0.1 - Genesis (current)
- [x] Procedural world with 9 biomes
- [x] 7 creature species with unique shapes
- [x] Autonomous AI behavior (wander, seek, work)
- [x] XP and leveling system
- [x] Mutations (horns, wings, body growth)
- [x] Task system with difficulty scaling
- [x] Particle effects
- [x] Camera panning and minimap
- [x] Creature inspection panel
- [x] Event log

## v0.2 - Bonds
- [ ] Creature breeding -- select two creatures to produce offspring
- [ ] Genetic crossover -- offspring inherit mixed stats from parents
- [ ] Trait dominance -- some mutations are dominant/recessive
- [ ] Family trees -- track lineage across generations
- [ ] Creature interactions -- creatures react to each other

## v0.3 - Persistence
- [ ] localStorage save/load -- creatures survive page refresh
- [ ] Export/import creature DNA as shareable codes
- [ ] World state persistence
- [ ] Creature history log

## v0.4 - Combat
- [ ] Arena battles -- creature vs creature
- [ ] Combat stats derived from base stats
- [ ] Battle animations
- [ ] Win/loss record tracking
- [ ] Tournament brackets

## v0.5 - Expansion
- [ ] More biomes (Ocean, Volcano, Tundra, Sky)
- [ ] More species
- [ ] More mutations (fins, armor, bioluminescence, extra limbs)
- [ ] Creature aging and natural death
- [ ] Rare legendary creatures

## v0.6 - Multiplayer
- [ ] WebSocket server for shared world state
- [ ] See other players creatures in real time
- [ ] Trade creatures between players
- [ ] Global leaderboards

## v1.0 - The Hatchery
- [ ] Sound design and ambient audio
- [ ] Mobile support with touch controls
- [ ] Token integration
- [ ] Creature marketplace
- [ ] Full persistent backend
